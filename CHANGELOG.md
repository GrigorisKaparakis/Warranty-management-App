# Changelog

Όλες οι σημαντικές αλλαγές σε αυτό το έργο θα καταγράφονται σε αυτό το αρχείο.

## [1.0.8] - 2026-03-27
### UI Animations & Refactoring
- **UI Micro-animations**: Προσθήκη staggered animations στη λίστα εγγυήσεων (Inventory), hover effects στις κάρτες και ομαλές μεταβάσεις (transitions) στα tabs του Maintenance Panel.
- **Button Interactions**: Προσθήκη hover και tap animations σε όλα τα κουμπιά της εφαρμογής για καλύτερη αίσθηση αλληλεπίδρασης.
- **Dashboard Animations**: Προσθήκη entrance animations στα στατιστικά και τα γραφήματα του Dashboard.
- **Refactoring useAppState**: Αναδιοργάνωση και καθαρισμός του κεντρικού hook της εφαρμογής για καλύτερη συντηρησιμότητα.

## [1.0.7] - 2026-03-26
### Βελτιστοποίηση Ρυθμίσεων & UI
- **Buffered Settings Updates**: Υλοποίηση τοπικού state (buffering) στις ρυθμίσεις της εφαρμογής (AppSettings). Οι αλλαγές πλέον αποθηκεύονται στη βάση δεδομένων μόνο μετά από ρητή επιβεβαίωση του χρήστη (Save button), μειώνοντας δραστικά τις εγγραφές στο Firestore και βελτιώνοντας την απόκριση του UI.
- **Dirty State Tracking**: Προσθήκη μηχανισμού ανίχνευσης αλλαγών ανά ενότητα ρυθμίσεων με δυνατότητα ακύρωσης (Cancel) και επαναφοράς των αρχικών τιμών.
- **UI Feedback**: Βελτιωμένη οπτική σήμανση για μη αποθηκευμένες αλλαγές στις ρυθμίσεις συντήρησης.

## [1.0.6] - 2026-03-26
### Αναδιοργάνωση & Βελτιστοποίηση Κώδικα
- **Refactoring useInventory**: Διάσπαση του hook σε `useInventoryFilters` και `useInventorySelection` για μείωση της πολυπλοκότητας και καλύτερη αναγνωσιμότητα.
- **Derived State Optimization**: Βελτιστοποίηση των υπολογισμών στατιστικών και λήξεων στο `useAppState` για αποφυγή πλεοναζόντων υπολογισμών.
- **Toast Utility Refactoring**: Πλήρης μετάβαση στη βιβλιοθήκη `sonner` και απλοποίηση του toast utility.
- **AdminService Cleanup**: Ενοποίηση της λογικής ενημέρωσης χρηστών και απλοποίηση της συγχώνευσης ρυθμίσεων (settings merging).
- **Config Cleanup**: Αφαίρεση διπλότυπων αντικειμένων defaults και χρήση του `FULL_GARAGE_DEFAULTS` ως μοναδική πηγή αλήθειας.
- **Code Cleanup**: Μαζική αφαίρεση αχρησιμοποίητων imports και εικονιδίων από όλη την εφαρμογή.
- **Pass-through State Removal**: Απευθείας χρήση του Zustand store σε components για μείωση του περιττού "prop drilling" μέσω του `useAppState`.

## [1.0.5] - 2026-03-24
### Βελτιστοποίηση Ταχύτητας
- Υλοποίηση `useDebounce` hook για την αναζήτηση στο Inventory.
- Μείωση των re-renders κατά την πληκτρολόγηση στην αναζήτηση, βελτιώνοντας την απόκριση του UI.
- Εφαρμογή `React.memo` στο `WarrantyCard` και σταθεροποίηση των callbacks (`useCallback`) για μέγιστη απόδοση σε μεγάλες λίστες.
- Ενσωμάτωση `persist` middleware στο Zustand store για τοπική αποθήκευση (localStorage) των Registries, των ρυθμίσεων και των εγγυήσεων, εξασφαλίζοντας ακαριαίο φόρτωμα της εφαρμογής.
- Δημιουργία `Global Toast Helper` (`src/utils/toast.ts`) για εύκολη εμφάνιση ειδοποιήσεων από οποιοδήποτε σημείο του κώδικα (components, services, hooks).
