# 📋 Λίστα Ελέγχου Εφαρμογής (Testing Checklist) - v1.2.0

Αυτή η λίστα περιλαμβάνει όλα τα σημεία ελέγχου για τη διασφάλιση της ορθής λειτουργίας της εφαρμογής **WARRANTY H&K**. Χρησιμοποιήστε την για manual testing πριν από κάθε νέα έκδοση.

---

## 🔐 1. Αυθεντικοποίηση & Προφίλ (Auth & Profile)
- [ ] **Google Login**: Σύνδεση με επιτυχία μέσω Google Popup.
- [ ] **Αρχικοποίηση Χρήστη**: Έλεγχος αν ένας νέος χρήστης παίρνει τον προεπιλεγμένο ρόλο (`USER`).
- [ ] **Logout**: Αποσύνδεση και ανακατεύθυνση στη σελίδα Login.
- [ ] **Προφίλ Χρήστη**:
    - [ ] Εμφάνιση σωστού Email.
    - [ ] Εμφάνιση σωστού Ρόλου (π.χ. ADMIN, EMPLOYEE).
    - [ ] Αλλαγή Πυκνότητας Εμφάνισης (Compact, Standard, Large) και άμεση εφαρμογή στο UI.
- [ ] **Απενεργοποίηση Χρήστη**: Έλεγχος αν ένας απενεργοποιημένος χρήστης (από τον Admin) χάνει την πρόσβαση.

## 📊 2. Dashboard (Πίνακας Ελέγχου)
- [ ] **Stats Cards**:
    - [ ] Σωστός υπολογισμός "ΣΥΝΟΛΟ ΕΓΓΥΗΣΕΩΝ".
    - [ ] Σωστός υπολογισμός "ΠΛΗΡΩΜΕΝΕΣ" (isPaid: true).
    - [ ] Σωστός υπολογισμός "ΣΕ ΑΝΑΜΟΝΗ" (Status: WAITING).
    - [ ] Σωστός υπολογισμός "ΑΠΟΡΡΙΦΘΗΚΑΝ" (Status: REJECTED).
- [ ] **Featured Statuses**: Εμφάνιση των καρτών για τις καταστάσεις που έχουν οριστεί στις ρυθμίσεις (π.χ. WAITING, WORKSHOP).
- [ ] **Γραφήματα**:
    - [ ] Κατανομή ανά Κατάσταση (Pie Chart).
    - [ ] Κατανομή ανά Εταιρεία (Bar Chart).
- [ ] **Πρόσφατη Δραστηριότητα**: Εμφάνιση των τελευταίων 5 (ή όσων ορίζονται) Audit Logs.
- [ ] **Ticker (Ανακοινώσεις)**:
    - [ ] Εμφάνιση κυλιόμενου κειμένου στην κορυφή.
    - [ ] Δυνατότητα προσθήκης νέας ανακοίνωσης (μόνο για ADMIN).
    - [ ] Δυνατότητα διαγραφής ανακοίνωσης.

## 📦 3. Inventory (Λίστα Εγγυήσεων)
- [ ] **Αναζήτηση**:
    - [ ] Λειτουργία αναζήτησης με VIN, Ονοματεπώνυμο, Warranty ID.
    - [ ] Έλεγχος Debounce (η αναζήτηση να ξεκινά μετά την καθυστέρηση που ορίζεται στα settings).
- [ ] **Φίλτρα**:
    - [ ] Φιλτράρισμα ανά Κατάσταση (Status).
    - [ ] Φιλτράρισμα ανά Εταιρεία (Company).
    - [ ] Φιλτράρισμα ανά Μάρκα (Brand).
    - [ ] Φιλτράρισμα ανά Ημερομηνία (Από - Έως).
    - [ ] Συνδυαστικά φίλτρα (π.χ. ΣΑΡΑΚΑΚΗΣ + WAITING).
- [ ] **Sorting**: Ταξινόμηση ανά Ημερομηνία, Warranty ID, Πελάτη, Μάρκα.
- [ ] **Επιλογή & Μαζικές Ενέργειες**:
    - [ ] Επιλογή μεμονωμένων εγγράφων.
    - [ ] "Επιλογή Όλων" στη σελίδα.
    - [ ] Μαζική αλλαγή Κατάστασης (Status).
    - [ ] Μαζική σήμανση ως Πληρωμένο/Απλήρωτο.
    - [ ] Μαζική Διαγραφή (με επιβεβαίωση).
- [ ] **Infinite Scroll**: Φόρτωση περισσότερων εγγραφών κατά το σκρολάρισμα.

## 📝 4. Detail View (Αναλυτική Προβολή)
- [ ] **Εμφάνιση Στοιχείων**: VIN, Πελάτης, Εταιρεία, Μάρκα, Ημερομηνία Καταχώρησης.
- [ ] **Status Management**: Αλλαγή κατάστασης από το dropdown και άμεση ενημέρωση.
- [ ] **Parts (Ανταλλακτικά)**:
    - [ ] Προσθήκη ανταλλακτικού (Κωδικός, Περιγραφή, Ποσότητα).
    - [ ] Suggestions: Εμφάνιση προτάσεων από το Registry κατά την πληκτρολόγηση κωδικού.
    - [ ] Σήμανση "Ready" (Πράσινο check) για κάθε ανταλλακτικό.
    - [ ] Διαγραφή ανταλλακτικού.
- [ ] **Notes (Σημειώσεις)**:
    - [ ] Προσθήκη νέας σημείωσης.
    - [ ] Εμφάνιση αυτόματου ιστορικού αλλαγών (System Notes).
- [ ] **Audit Timeline**: Εμφάνιση της κάθετης γραμμής χρόνου με όλες τις ενέργειες που αφορούν τη συγκεκριμένη εγγύηση.
- [ ] **Manual Expiry Override**:
    - [ ] Χειροκίνητη αλλαγή ημερομηνίας λήξης.
    - [ ] Καθαρισμός ημερομηνίας λήξης (null).
    - [ ] Οπτική σήμανση αν η εγγύηση έχει λήξει (Alert icon).
- [ ] **Εκτύπωση/PDF**: Παραγωγή PDF με τα στοιχεία της εγγύησης και τα ανταλλακτικά.

## 📑 6. Registries (Μητρώα)

- [ ] **Vehicle Registry**: Αναζήτηση VIN, εμφάνιση τελευταίου ιδιοκτήτη.
- [ ] **Customer Registry**: Αναζήτηση πελάτη, εμφάνιση συνδεδεμένων VIN.

## 📜 7. Audit Logs (Καθολικό Ιστορικό)
- [ ] **Εμφάνιση**: Λίστα με όλες τις ενέργειες του συστήματος.
- [ ] **Φίλτρα**: Αναζήτηση ανά Warranty ID, Χρήστη, Ενέργεια (CREATE, UPDATE, κλπ).
- [ ] **Details**: Προβολή "Πριν" και "Μετά" για κάθε αλλαγή.
- [ ] **Restore**: Επαναφορά διαγραμμένης εγγύησης ή προηγούμενης κατάστασης.

    - [ ] Σωστή ανάλυση ανά Εταιρεία.
